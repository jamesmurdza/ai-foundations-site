Build intelligent Minecraft bots — from simple followers to Claude-controlled agents. This course takes you from creating your first Minecraft bot to controlling it with Claude through MCP (Model Context Protocol).

## What you'll build

| #   | Tutorial                                                  | What you'll learn                                       |
| --- | --------------------------------------------------------- | ------------------------------------------------------- |
| 1   | [Bot Basics](/courses/minecraft-ai/tutorial-1-bot-basics) | Create a bot, connect to a server, follow a player      |
| 2   | [Smart Bot](/courses/minecraft-ai/tutorial-2-smart-bot)   | Add combat, coordinate navigation, and state management |
| 3   | [MCP Server](/courses/minecraft-ai/tutorial-3-mcp-server) | Control your bot with Claude through MCP                |

## Prerequisites

Make sure you've completed the [Setup Guide](/courses/minecraft-ai/setup) before starting — you'll need Node.js, Java, a PaperMC server, a Minecraft client, and Claude Desktop or Claude Code.
